Node.js 16.x以上がインストールされているか確認\n   - npm install を実行\n   - ログファイル確認 (logs/start-*.log)\n\n2. **フレーム表示エラー**\n   - ファイル形式が対応しているか確認\n   - ファイルサイズ制限（500MB以下）\n   - メモリ使用量確認\n\n3. **エクスポート失敗**\n   - 書き出し先ディレクトリの権限確認\n   -
